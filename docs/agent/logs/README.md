# Agent Session Logs

This directory stores short iteration logs to prevent repeated investigations
and to keep agent handoffs efficient.

## Rules

- One file per session or coherent iteration.
- Use filename format: `YYYY-MM-DD-short-title.md`.
- Keep entries concise and factual.
- Link each session file from `index.md`.

## Required Sections per Session

- Objective
- Changes Applied
- Validation
- Open Risks
- Next Step

Use `log-template.md` for consistency.
